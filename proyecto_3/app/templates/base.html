Manual del Administrador

Sistema de Gestión de Inventario — Tienda El Despecho. Consulta cada sección o descarga el manual completo en PDF.

1. Acceso al Sistema
  • Abrir el navegador y dirigirse a http://localhost:8000.
  • Ingresar usuario y contraseña y hacer clic en Iniciar sesión.
  • Para cerrar sesión usar el botón Salir en la barra superior.
  • Si olvidaste tu contraseña, usa el enlace ¿Olvidaste tu contraseña?; recibirás un código al correo.
2. Panel Principal (Dashboard)
  • Tarjetas de resumen: totales de productos, clientes, ventas, proveedores, compras y reportes.
  • Gráficas: ventas e ingresos por mes, top 5 productos más vendidos, estado de clientes.
  • Alertas de stock bajo: productos con 10 o menos unidades aparecen destacados.
  • Asistente IA: botón flotante en la esquina inferior derecha.
3. Módulo de Productos
  • Agregar: clic en + Agregar Producto, completar nombre, precio, stock, marca, tipo y unidad.
  • Editar: ícono de lápiz en la fila del producto.
  • Eliminar: ícono de basura (pide confirmación).
  • Código de barras: se puede registrar el código EAN al crear o editar el producto.
  • Desde este módulo también se gestionan Marcas, Tipos y Unidades de Medida.
4. Módulo de Ventas
  • Nueva venta manual: clic en + Nueva Venta, buscar cliente, agregar productos y confirmar.
  • Venta con escáner: clic en Escanear venta y apuntar la cámara al código de barras.
  • Completar: una venta Pendiente se completa haciendo clic en .
  • Exportar: botones PDF y Excel en la parte superior de la tabla.
5. Módulo de Compras
  • Registra compras a proveedores indicando producto, cantidad, precio unitario y proveedor.
  • El stock del producto se actualiza automáticamente al completar la compra.
6. Clientes y Proveedores
  • CRUD completo: crear, editar, eliminar y cambiar estado (activo/inactivo) para clientes.
  • En Proveedores se pueden asignar productos haciendo clic en el badge azul.
7. Escáner de Códigos de Barras
  • Modo inventario: Catálogo → Productos → Escanear. Actualiza stock.
  • Modo venta: Ventas → Escanear venta. Agrega al carrito.
  • También se puede cargar una imagen con el código de barras.
8. Reportes y Exportación
  • Vista general en Sistema → Reportes con estadísticas globales.
  • Cada módulo tiene botones Exportar PDF y Exportar Excel.
9. Gestión de Usuarios
  • Solo visible para el superadministrador.
  • Sistema → Usuarios → Crear Administrador.
10. Respaldo y Restauración
  • Sistema → Respaldo BD → Descargar SQL para generar una copia de seguridad.
  • Para restaurar, arrastra el archivo .sql al área de restauración.
  • Advertencia: restaurar sobrescribe todos los datos actuales.
